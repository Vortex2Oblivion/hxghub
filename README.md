### hxghub

![](https://raw.githubusercontent.com/Vortex2Oblivion/hxghub/master/assets/hxpy-logo.png)

Haxe/hxcpp @:native bindings for Logitech GHUB

This library works on the C++ target only!

### Installation

This library is only avaliable via git as of now
```bash
haxelib git hxpy https://github.com/Vortex2Oblivion/hxghub.git
```
